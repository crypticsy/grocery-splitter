## 📋 Requirements
To run this project by yourself, you need:
- Python 3.7+

Install the dependencies with:

```bash
pip install -r requirements.txt
```

<br>

## 🖥️ Usage

```shell
# open the streamlit server
streamlit run Home.py
```
